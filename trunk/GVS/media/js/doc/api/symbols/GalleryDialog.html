GalleryDialog($super, title, _properties)
This class handles dialogs with a gallery (a table) of elements Any descendant must use this class as follows: It must call to _setFields, _setButtons and _addRows, before calling _render, which will be the method that builds the interface, using the information that has been stated in the previous methods.
